## 📞 Support

### Common Questions

**Q: Do I need to install anything?**  
A: No! Open in browser - 100% works in the browser.

**Q: Will my files be uploaded?**  
A: No! All processing is on your device - completely private.

**Q: What if social media download doesn't work?**  
A: Some platforms restrict downloads. Use external tools like yt-dlp or youtube-dl.

**Q: First time using video/image tool is slow?**  
A: Yes, libraries download ~30MB on first use. That's normal.
