# 📚 SentinelConvert - Complete Feature Guide (v3.0)

## ⚙️ Technical Specifications

### Processing
- **100% Client-Side:** All processing happens in your browser
- **No Server Uploads:** Files never leave your device
- **No Data Collection:** Complete privacy (except Google AdSense)
- **Offline Capable:** Works without internet (except for CDN libraries on first load)

### Browser Support
- Chrome/Edge (v90+)
- Firefox (v88+)
- Safari (v14+)
- Opera (v76+)

### File Size Limits
- Practical limit: Up to 500MB (depends on device RAM)
- Most files under 100MB work smoothly

### Performance
- PDF merge/split: <1s for typical files
- Image conversion: 1-3s per image
- Video conversion: Varies by file size (usually minutes)
- Utilities (JSON, QR, Base64): Instant

## 🚀 Getting Started

### First Visit
1. Open SentinelConvert in your browser
2. Select a tool from the grid
3. Follow the on-screen instructions
4. Download your result

### Keyboard Shortcuts
- **Search box:** Type to find tools quickly
- **Category filter:** Click category headers
- **Copy output:** Ctrl+C (where available)

### Tips & Tricks
- For large PDFs, split first then process
- Batch processing: Process multiple files at once
- Social media downloads: May need external tools for some platforms
- AI background removal: Works best with clear backgrounds
- Video download: Direct links work better than platform links

## ⚠️ Limitations & Known Issues

### Social Media Downloading
- YouTube: May be restricted (use yt-dlp, youtube-dl as alternatives)
- TikTok: May need proxy
- Instagram: May need authentication
- Facebook: Limited support
- **Recommendation:** For guaranteed support, use external command-line tools

### Video Processing
- Very large files (>1GB) may slow down your browser
- First FFmpeg use downloads ~30MB
- Some rare video codecs may not be supported

### Format-Specific Notes
- PPTX to PDF: Complex animations and advanced formatting may be lost
- PDF to PPTX: Pages rendered as images (not editable text)
- DOCX parsing: Some formatting may be simplified
- Background removal: Works best on clear backgrounds

## 🔧 Troubleshooting

### Issue: "File too large"
**Solution:** Split the file first or use a device with more RAM

### Issue: Conversion fails
**Solution:** Try again - sometimes CDN libraries need reloading

### Issue: Video won't download
**Solution:** Check if URL is direct link or platform link. Use external tools for platform videos.

### Issue: Tools not loading
**Solution:** 
1. Refresh the page
2. Clear browser cache
3. Try a different browser
4. Check internet connection (for CDN libraries)
